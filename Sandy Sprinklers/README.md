# Sandy Sprinklers

Sandy Sprinklers allows sprinklers to be placed on sand tiles at the Beach, Ginger Island, and Beach Farm.

By default, Stardew Valley prevents sprinkler placement on these tiles even though they visually resemble farmland. This mod relaxes that restriction without affecting normal farm dirt or crop placement elsewhere.

## Features
- Place sprinklers on beach sand
- Works at the Beach, Ginger Island, and Beach Farm
- No Harmony required
- Does not change planting rules on normal farm tiles
- Multiplayer safe
- Stardew Valley 1.6+

## Requirements
- SMAPI 4.0 or newer

## Installation
1. Install SMAPI.
2. Extract the `SandySprinklers` folder into your `Mods` directory:
   - **Steam / GOG:** `Stardew Valley/Mods`
   - **Xbox PC:** `C:\XboxGames\Stardew Valley\Content\Mods`
3. Launch the game using `StardewModdingAPI.exe`.

## Usage
Hold a sprinkler and place it on sand using the normal left or right mouse click.

## Uninstall
Remove the `SandySprinklers` folder from your `Mods` directory.

## Compatibility
Sandy Sprinklers is compatible with most mods. Any mod that heavily alters map tile properties may affect where sprinklers can be placed.
